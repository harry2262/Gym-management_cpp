

# Gym Member Management System

This is a C++ program designed to manage members of a gym. The program allows the user to add members, display member information, view all member data, and view the total number of members.

## Features

- Add new members to the gym
- Display member information
- View all member data
- View the total number of members

## Usage

The program runs in a console or command-line interface. The user can interact with the program through a series of text-based menus and prompts, entering data and commands using the keyboard.

To use the program, compile and run it using a C++ compiler. Follow the on-screen prompts to perform various operations.

## Data Storage

The program stores data in a file, allowing for easy retrieval and updating of information. The data is stored in a file named `ww.txt`, which stores information about the gym's members.
